# AI员工管理系统

> 让创业者用AI员工替代真人团队，48小时内搭建自己的数字公司，成本降低98%

## 🚀 快速开始

```bash
# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入你的 API Keys

# 运行演示
python demo.py

# 或运行完整系统
python main.py
```

## 🛠️ 技术栈

- **前端**: Next.js 14 + Tailwind CSS + shadcn/ui
- **后端**: Vercel Serverless + Supabase
- **AI**: DeepSeek API (主力) + OpenAI (备用)
- **支付**: Lemon Squeezy
- **分析**: Plausible
