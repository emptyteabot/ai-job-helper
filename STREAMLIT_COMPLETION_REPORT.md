## 🔗 API 集成

### 后端 API 地址
```
https://ai-job-hunter-production-2730.up.railway.app
```

### 使用的端点

1. **简历分析**
   - `POST /api/process`
   - 支持文件上传和文本输入
   - 返回 6 大 AI 分析结果

2. **自动投递**
   - `POST /api/auto-apply/start-multi` - 启动多平台投递
   - `POST /api/auto-apply/stop` - 停止投递
   - `GET /api/auto-apply/status/{task_id}` - 查询状态
   - `GET /api/auto-apply/history` - 查询历史

## 🚀 如何运行

### 本地运行

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动应用
streamlit run streamlit_app.py

# 3. 访问
浏览器打开 http://localhost:8501
```

### Streamlit Cloud 部署

1. Fork 仓库到你的 GitHub
2. 访问 https://streamlit.io/cloud
3. 连接 GitHub 仓库
4. 选择 `streamlit_app.py`
5. 点击 Deploy
