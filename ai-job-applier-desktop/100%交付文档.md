## 🚀 完整使用流程

### 步骤 1：上传简历
1. 访问：https://ai-job-helper.streamlit.app
2. 点击「简历分析」页面
3. 上传简历 PDF 或粘贴简历文本
4. 点击「开始分析」
5. 等待 4 位 AI 专家分析完成

### 步骤 2：登录 Boss 直聘
1. 切换到「自动投递」页面
2. 输入你的手机号（11 位）
3. 点击「登录 Boss 直聘」
4. **会弹出浏览器窗口**（这是正常的）
5. 在浏览器中：
   - 输入短信验证码
   - 点击登录
6. 登录成功后，浏览器会自动关闭
7. 返回网站，显示「已登录」

### 步骤 3：自动投递
1. 填写信息：
   - 想找什么工作？（如：Python 开发）
   - 在哪个城市？（如：北京）
   - 投递多少个？（建议 10 个）
   - 你的邮箱（可选）
2. 点击「开始自动投递」
3. 等待投递完成（约 2-5 分钟）
4. 查看投递结果


## 🐛 常见问题

### Q1: 点击登录后没有弹出浏览器？
**A**: 检查后端服务是否正在运行：
```bash
netstat -ano | findstr :8765
```
如果没有输出，说明后端没有运行，需要重新启动。

### Q2: 登录超时怎么办？
**A**: 
1. 检查网络连接
2. 确保 Cloudflare Tunnel 正在运行
3. 重新点击登录

### Q3: 投递失败怎么办？
**A**: 
1. 检查是否已登录 Boss 直聘
2. 检查网络连接
3. 查看后端日志：`backend/backend.log`

### Q4: 如何重启服务？
**A**: 
```bash
# 1. 停止后端
taskkill /F /PID <进程ID>

# 2. 启动后端
cd backend
python main.py --port 8765

# 3. 确保 Cloudflare Tunnel 正在运行
cloudflared tunnel --url http://localhost:8765
```
