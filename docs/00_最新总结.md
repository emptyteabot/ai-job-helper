# 最新总结（以当前代码为准）

## 现在真正能用的 MVP

1. **AI 分析简历（DeepSeek）**
- 页面：`/app`
- 接口：`POST /api/process`
- 需要：`DEEPSEEK_API_KEY`

2. **本机实时搜 Boss 岗位链接（手动投递）**
- 页面：结果区的“实时岗位搜索（Boss直聘）”
- 数据源：`JOB_DATA_PROVIDER=openclaw`
- 机制：OpenClaw 控制你已登录的 Chrome（需要你手动 Attach）
- 产物：`job_detail` 链接列表，点击后你手动投递

3. **投递记录**
- 后端会记录一次“跳转投递”的记录（不会假装已自动提交）

## 云端 24h 怎么做

- 云端可以部署“AI 分析 Web 服务”，但不要启用 OpenClaw。
- OpenClaw 只适用于本机，因为它依赖你的浏览器登录态。
