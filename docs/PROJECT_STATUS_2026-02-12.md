## 2. 已确认问题与根因

### 2.1 GitHub Actions 报 `on/jobs is already defined`
- 根因：workflow 文件之前出现重复 YAML 顶层键。
- 当前仓库状态：`.github/workflows/deploy.yml` 已是单一结构（可继续观察下次运行）。

### 2.2 OpenClaw 在 Railway 不稳定
- 根因：OpenClaw依赖本地浏览器扩展 attach，不适合无头云环境长期在线。
- 处理策略：云端默认走无浏览器可用的数据源；OpenClaw仅作为本地增强模式。

### 2.3 乱码/问号现象
- 根因通常是编码链路或客户端解码差异。
- 当前处理：前端主视觉已改为英文极简，岗位核心信息通过结构化卡片渲染，降低乱码感知。
