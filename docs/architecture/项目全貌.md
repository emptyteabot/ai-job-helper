## 🤖 核心功能模块

### 1. 多AI协作引擎 (`multi_ai_debate.py`)

**功能**: 6个AI角色协作，互相辩论改进

```python
AI角色列表:
├─ AI-1: 职业规划师 (career_planner)
│   └─ 任务: 分析优势、定位职业方向
│
├─ AI-2: 招聘专家 (recruiter)
│   └─ 任务: 搜索匹配岗位
│
├─ AI-3: 简历优化师 (resume_optimizer)
│   └─ 任务: 改写简历内容
│
├─ AI-4: 质量检查官 (quality_checker)
│   └─ 任务: 审核简历，指出问题
│
├─ AI-5: 面试教练 (interview_coach)
│   └─ 任务: 提供面试辅导
│
└─ AI-6: 模拟面试官 (interviewer)
    └─ 任务: 模拟真实面试
```

**协作流程**:
```
用户简历 
  → AI-1 分析优势
  → AI-2 推荐岗位
  → AI-3 优化简历
  → AI-4 审核检查
  → AI-3 再次优化
  → AI-5 面试辅导
  → AI-6 模拟面试
  → 输出结果
```

**关键方法**:
- `ai_think()` - 单个AI思考
- `debate_chain()` - AI辩论链
- `_clean_markdown()` - 清理格式

### 2. 文件上传解析 (`web_app.py` - `/api/upload`)

**支持格式**:
- ✅ PDF文件 (.pdf) - PyPDF2解析
- ✅ Word文档 (.docx, .doc) - python-docx解析
- ✅ 文本文件 (.txt) - 直接读取
- ✅ 图片文件 (.jpg, .png等) - OCR识别

**解析流程**:
```
上传文件
  → 检查格式
  → 读取内容
  → 根据格式解析
    ├─ PDF: PyPDF2提取文本
    ├─ Word: python-docx提取段落+表格
    ├─ TXT: 直接解码
    └─ 图片: Tesseract OCR识别
  → 返回文本内容
  → 自动开始AI处理
```

### 4. 岗位搜索 (`job_searcher.py`)

**功能**: 搜索匹配岗位（当前为模拟数据）

**匹配算法**:
```python
匹配度 = 技能匹配度(70%) + 职位意向匹配(30%)

技能匹配度 = 匹配的技能数 / 岗位要求技能数
```

**模拟岗位数据**:
- 字节跳动 - Python后端开发
- 阿里巴巴 - 全栈开发
