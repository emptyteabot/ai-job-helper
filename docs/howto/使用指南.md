# AI求职助手 - 多AI协作系统

## 🎯 核心功能

这是一个**多AI协作辩论系统**，让多个AI像团队一样协作，互相改进输出：

```
用户上传简历 
    ↓
AI-1 职业规划师：分析优势
    ↓
AI-2 招聘专家：搜索岗位
    ↓
AI-3 简历优化师：改写简历 → AI-4 质量检查官：审核 → AI-3 再次优化
    ↓
自动投递
    ↓
AI-5 面试教练：辅导准备 → AI-6 模拟面试官：测试
```

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install openai python-dotenv
```

### 2. 配置API密钥

```bash
set DEEPSEEK_API_KEY=你的DeepSeek API密钥
```

### 3. 运行测试

**方式1：双击运行**
```
双击 快速测试.bat
```

**方式2：命令行运行**
```bash
python 快速测试.py
```

## 📋 完整流程

### 输入
- 上传简历（PDF/Word/文本）

### 处理流程
1. **简历分析** - 提取技能、经验、项目
2. **岗位搜索** - 从各大招聘网站搜索匹配岗位
3. **多AI协作优化简历**：
   - AI-1 职业规划师：分析优势和定位
   - AI-2 招聘专家：推荐最匹配岗位
   - AI-3 简历优化师：针对岗位改写简历
   - AI-4 质量检查官：审核并要求改进
   - AI-3 再次优化：根据反馈完善
4. **自动投递** - 一键投递到各大平台
5. **面试准备**：
   - AI-5 面试教练：提供辅导建议
   - AI-6 模拟面试官：模拟真实面试

### 输出
- ✅ 职业分析报告
- ✅ 推荐岗位列表（匹配度排序）
- ✅ 优化后的简历（针对每个岗位）
- ✅ 面试准备指南
- ✅ 模拟面试问答
- ✅ 完整AI辩论记录

## 🤖 多AI协作机制

### 核心思想
让多个AI像辩论一样协作：
- **AI-1 输出** → **AI-2 改进** → **AI-3 检查** → **AI-2 再优化**
- 每个AI都有专门的角色和任务
- 后面的AI基于前面的输出进行改进
- 形成"辩论-改进-审核"的闭环

### 6个AI角色

| 角色 | 任务 | 特点 |
|------|------|------|
| 职业规划师 | 分析优势、定位方向 | 战略眼光 |
| 招聘专家 | 搜索匹配岗位 | 行业洞察 |
| 简历优化师 | 改写简历内容 | 文案能力 |
| 质量检查官 | 审核并要求改进 | 严格标准 |
| 面试教练 | 提供面试辅导 | 实战经验 |
| 模拟面试官 | 模拟真实面试 | 压力测试 |

## 🎨 特色功能

### 1. Markdown格式清理
自动清理DeepSeek输出中的 `*`、`**`、`###` 等符号，让结果更干净易读。

### 2. 智能匹配算法
根据技能、经验、意向计算岗位匹配度，优先推荐最合适的岗位。

### 3. 完整辩论记录
保存所有AI的思考过程和推理链，方便回溯和优化。

### 4. 模块化设计
每个功能独立模块，易于扩展和维护。
