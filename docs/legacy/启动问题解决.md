# 🚀 快速启动指南

## 问题：网页进不去

### 可能的原因：
1. ❌ 依赖未安装成功
2. ❌ 服务未启动
3. ❌ 端口被占用
4. ❌ 浏览器缓存问题

---

## 🎯 解决方案

### 方法1: 使用CMD直接运行（推荐）

#### 步骤1: 打开CMD
按 `Win + R`，输入 `cmd`，回车

#### 步骤2: 进入项目目录
```cmd
cd C:\Users\陈盈桦\Desktop\自动投简历
```

#### 步骤3: 安装依赖
```cmd
pip install fastapi uvicorn websockets python-multipart python-dotenv openai
```

#### 步骤4: 启动服务
```cmd
python web_app.py
```

#### 步骤5: 访问网页
打开浏览器，访问: http://localhost:8000

---

### 如果看到错误：

#### 错误1: ModuleNotFoundError: No module named 'fastapi'
**解决**: 依赖未安装
```cmd
pip install fastapi uvicorn websockets
```

#### 错误2: Address already in use
**解决**: 端口被占用
```cmd
# 方法A: 换个端口
set PORT=8001
python web_app.py

# 方法B: 关闭占用端口的程序
netstat -ano | findstr :8000
taskkill /PID [进程ID] /F
```

#### 错误3: 网页显示"无法访问此网站"
**解决**: 
1. 确认服务已启动（看到上面的成功信息）
2. 检查浏览器地址是否正确: http://localhost:8000
3. 尝试使用 http://127.0.0.1:8000
4. 清除浏览器缓存（Ctrl+Shift+Delete）

## 💡 常见问题

### Q1: 双击bat文件一闪而过
**A**: 这是正常的，说明脚本执行完了。请使用CMD手动运行。

### Q2: 提示"不是内部或外部命令"
**A**: Python未安装或未添加到PATH。重新安装Python，勾选"Add to PATH"。

### Q3: pip安装很慢
**A**: 使用国内镜像源：
```cmd
pip install fastapi -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### Q4: 浏览器显示404
**A**: 
- 确认访问的是 http://localhost:8000 或 http://localhost:8000/app
- 不要访问 http://localhost:8000/index.html
