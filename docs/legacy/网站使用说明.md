## 🔧 常见问题

### Q: 如何停止服务？
**A:** 在命令行窗口按 `Ctrl+C`

### Q: 端口被占用怎么办？
**A:** 修改 `web_app.py` 最后一行的端口号：
```python
uvicorn.run(app, host="0.0.0.0", port=8001)  # 改成8001
```

### Q: 可以在手机上访问吗？
**A:** 可以！在同一WiFi下，用手机浏览器访问：
```
http://你的电脑IP:8000
```

### Q: 结果保存在哪里？
**A:** 
1. 网页上可以直接查看
2. 点击"下载结果"保存到本地
3. 也会自动保存到 `output/` 目录
