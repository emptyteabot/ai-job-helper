# 11 Marketing GTM Playbook

更新时间：2026-02-17
Owner：Marketing / GTM

## 1. 定位

- 一句话：上传简历，快速拿到可执行的投递方案。
- 核心价值：更快投递、更高命中、更少无效尝试。

## 2. 渠道优先级（中国市场）

- 第一梯队：小红书、B站、微信群/社群
- 第二梯队：知乎、公众号、校招社群
- 第三梯队：合作渠道（训练营/高校）

## 3. 增长漏斗

- 曝光 -> 访问 -> 上传 -> 分析成功 -> 岗位点击 -> 投递 -> 复访

## 4. 内容策略

- 内容类型 A：真实用户前后对比
- 内容类型 B：岗位匹配与简历优化案例
- 内容类型 C：面试问题拆解与复盘

## 5. 周实验节奏

每周至少 3 个实验：
- 文案实验（首页标题/CTA）
- 转化实验（上传流程）
- 留存实验（复访触发）

## 6. 指标目标（首月）

- 访问->上传 >= 20%
- 上传->分析成功 >= 85%
- 分析->岗位点击 >= 25%
- 周复访率 >= 15%

## 7. 市场化进入条件

- 技术侧发布闸门通过
- 真实岗位返回率达到门槛
- 客诉率可控（P0/P1）
