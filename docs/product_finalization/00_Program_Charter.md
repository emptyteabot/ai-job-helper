# Program Charter - 最终产品化计划

## 1. 目标

在 4 周内把 AI 求职助手从“技术可用”推进到“市场可卖”：

- 对用户：上传简历后可获得真实、可点击、可行动的岗位建议
- 对运营：具备可追踪漏斗和稳定上线能力
- 对商业：具备 GTM 叙事、试用转化路径和定价实验框架

## 2. 产品范围

In Scope:

- 简历解析、职业分析、岗位推荐、简历优化、面试准备
- 中国招聘市场优先数据链路
- Web 端极简体验和核心分析流程

Out of Scope:

- 完整 ATS 系统
- 企业组织管理后台
- 跨区域多语言深度本地化

## 3. 成功标准

- 业务指标：上传到分析转化率 >= 60%
- 数据指标：推荐岗位可点击率 >= 80%
- 技术指标：P95 API 响应 <= 3.5s（非长任务接口）
- 稳定性：7 天无阻断级事故

## 4. 阶段里程碑

- M1（Week1）: 需求冻结 + 架构冻结 + 设计冻结
- M2（Week2）: 核心功能完成 + 接口联调通过
- M3（Week3）: 全量回归 + 压测 + UAT
- M4（Week4）: 发布上线 + Marketing Launch

## 5. Go/No-Go 闸门

Go 条件:

- P0 缺陷为 0
- 关键漏斗可追踪
- 中国岗位数据链路在云端可持续可用
- 发布回滚方案和 on-call 值守就绪

No-Go 条件:

- 推荐岗位为空或不可点击占比 > 20%
- 核心路径（上传->分析）失败率 > 5%

## 6. 沟通机制

- 每日 15 分钟站会
- 每周一次跨团队评审
- 所有需求变更必须走变更单
