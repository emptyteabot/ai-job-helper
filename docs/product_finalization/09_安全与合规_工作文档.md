# 安全与合规工作文档

## 角色目标

在不牺牲体验的前提下，控制安全和合规风险。

## 范围

- 简历数据处理安全
- API 密钥管理
- 第三方依赖与外部请求风险
- 日志脱敏

## 基线要求

- 环境变量管理敏感配置
- 禁止在日志记录完整个人隐私信息
- 明确数据保存周期和删除策略

## 合规要求

- 用户可知晓数据用途
- 数据处理边界清晰
- 对外分享前做最小化处理

## 风险控制

- 外部接口异常不泄露内部细节
- 上传文件格式和大小限制
- 依赖升级有安全审查

## 协作接口

- 与产品确定隐私告知文案
- 与后端确定脱敏规则
- 与 DevOps 审查密钥和访问控制
