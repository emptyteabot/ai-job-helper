# 数据分析与增长工作文档

## 角色目标

建立“可度量增长系统”，让产品优化有数据闭环。

## 指标体系

北极星指标:

- 每周有效分析用户数（完成分析并点击岗位）

核心漏斗:

- 访问首页 -> 进入工作台
- 进入工作台 -> 提交分析
- 提交分析 -> 点击岗位链接
- 点击岗位链接 -> 下载结果

## 事件埋点规范

- 统一事件命名
- 统一 user/session 标识
- 统一关键字段：provider_mode, result_count, latency

## 实验体系

- A/B 测试：CTA 文案、结果布局、岗位卡片结构
- 每周至少 1 个实验
- 每个实验有明确停止条件

## 数据交付

- 周报：漏斗转化 + 异常说明
- 月报：增长贡献拆解 + 下月策略

## 协作接口

- 与产品定义指标口径
- 与前后端对齐埋点实现
- 与 marketing 对齐投放归因
