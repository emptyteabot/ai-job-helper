# 增长周报自动化

## 目标

把 marketing 阶段核心漏斗数据自动产出为周报，避免手工整理。

## 脚本

- `scripts/generate_weekly_growth_report.py`

## 用法

默认读取 `data/app_data.db`，输出到 `docs/reports/`:

```bash
python scripts/generate_weekly_growth_report.py
```

指定天数和输出目录:

```bash
python scripts/generate_weekly_growth_report.py --days 14 --out-dir docs/reports
```

## 输出内容

- 漏斗总览（upload/process/search/apply）
- 转化率指标
- 每日事件表
- 下周动作建议

## 并行协作约定

- 数据分析负责解读周报
- 产品负责下一周实验计划
- Marketing 负责渠道动作和内容节奏
