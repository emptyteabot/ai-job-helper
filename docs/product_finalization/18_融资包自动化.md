# 融资包自动化

## 目标

自动产出投资人汇报基础材料，减少手工整理时间。

## 脚本

- `scripts/generate_investor_pack.py`

## 用法

```bash
python scripts/generate_investor_pack.py
```

输出:

- `docs/investor/investor_pack_YYYY-MM-DD.md`
- `docs/investor/investor_pack_YYYY-MM-DD.json`

## 数据来源

- 投资人 readiness 快照（产品/增长/可靠性/GTM 四支柱）
- 业务漏斗指标与稳定性指标
- 30 天目标建议

## 融资周会流程

1. 先看 `overall_score` 和四支柱变化
2. 再看漏斗短板（upload->process->search->apply）
3. 最后确定下一周动作和负责人
