# 商业化运营接口说明

## 新增能力
- 留资收集：`POST /api/business/lead`
- 漏斗指标：`GET /api/business/metrics`
- 融资就绪度：`GET /api/business/readiness`

## 鉴权
- 可选环境变量：`ADMIN_METRICS_TOKEN`
- 设置后，请在请求头带：`x-admin-token: <token>`

## 指标定义
- `uploads`：简历上传成功次数
- `process_runs`：简历分析请求次数（成功/失败都计）
- `searches`：岗位搜索次数
- `applies`：投递动作次数
- `upload_to_process_pct`：上传 -> 分析转化率
- `process_to_search_pct`：分析 -> 搜索转化率
- `search_to_apply_pct`：搜索 -> 投递转化率

## 示例
```bash
curl https://ai-job-hunter-production-2730.up.railway.app/api/business/metrics
```

```bash
curl -X POST https://ai-job-hunter-production-2730.up.railway.app/api/business/lead \
  -H "Content-Type: application/json" \
  -d '{"email":"you@company.com","name":"张三","source":"landing"}'
```

## 建议运营动作
1. 把首页留资表单作为默认 CTA，先累计第一批高意向线索。
2. 每周导出一次 metrics，追踪漏斗提升。
3. 当 `search_to_apply_pct` 连续两周 > 20% 时，启动付费试点。
