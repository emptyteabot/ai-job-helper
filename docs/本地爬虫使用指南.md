## 🚀 现在开始使用

### 第一步：配置OpenClaw（一次性，5分钟）

#### 1. 启动OpenClaw浏览器

打开命令行，运行：
```bash
openclaw browser start
```

这会打开一个受OpenClaw控制的Chrome浏览器。

#### 2. 登录Boss直聘

在打开的Chrome中：
1. 访问：https://www.zhipin.com
2. 扫码或密码登录您的Boss直聘账号
3. 保持登录状态

#### 3. 连接OpenClaw扩展

1. 点击Chrome右上角的 **OpenClaw扩展图标**（🔧拼图图标）
2. 在弹出的面板中点击 **"Attach"** 按钮
3. 看到 "✅ Connected" 提示即表示成功

**重要：**
- ✅ 连接一次后，只要不关闭浏览器就一直有效
- ✅ 可以最小化浏览器窗口
- ⚠️ 不要关闭浏览器

---

#### 方式1：使用启动脚本（推荐）

```

#### 方式2：命令行运行

cd "c:\Users\陈盈桦\Desktop\Desktop_整理_2026-02-09_172732\Folders\自动投简历"

## 🔧 自定义配置

### 修改爬取关键词

编辑 `openclaw_crawler_service.py`：

```python
# 第30行左右
self.hot_keywords = [
    ["您想要的关键词1"],
    ["您想要的关键词2", "相关词"],
    # 添加更多...
]
```

### 修改城市

```python
# 第42行左右
self.hot_cities = ["北京", "上海", "您想要的城市"]
```

### 修改爬取频率

编辑 `crawler.env`：
```env
CRAWL_INTERVAL_HOURS=2  # 改为2小时
```

## 🛠️ 常见问题

### Q1: OpenClaw连接失败？

**症状：**
```
❌ OpenClaw 浏览器未连接标签页
```

**解决：**
1. 运行 `openclaw browser start`
2. 在Chrome中打开Boss直聘
3. 点击OpenClaw扩展图标
4. 点击"Attach"按钮

### Q2: 推送失败？

**症状：**
```
❌ 推送失败：401 - 未授权
```

**解决：**
检查 `crawler.env` 中的 `CRAWLER_API_KEY` 是否正确

### Q3: 爬取到0个岗位？

**原因：**
- Boss直聘页面未加载完成
- OpenClaw未正确连接
- 搜索关键词太精确

**解决：**
1. 等待几秒后重试
2. 检查OpenClaw连接状态
3. 调整搜索关键词

### Q4: 如何停止爬虫？

按 `Ctrl+C` 停止服务

### Q5: 如何查看爬虫日志？

所有输出都在控制台显示，可以重定向到文件：
```bash
python openclaw_crawler_service.py > crawler.log 2>&1
```
