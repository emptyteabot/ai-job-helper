## 🚀 使用步骤

### 方式1：上传文件（推荐）

1. **双击运行**
   ```
   双击 测试上传功能.bat
   ```

2. **打开浏览器**
   ```
   访问 http://localhost:8000
   ```

3. **找到上传按钮**
   - 在页面中间偏上的位置
   - 有个大按钮写着：**📎 点击上传简历文件（支持PDF、Word、TXT）**
   - 按钮是灰色的，很大，占满整行

4. **点击上传**
   - 点击按钮
   - 选择您的简历文件（PDF/Word/TXT）
   - 等待几秒钟
   - 看到 "✅ 文件上传成功：文件名.pdf"
   - 简历内容自动填入下面的文本框

5. **开始AI协作**
   - 点击 "🚀 开始AI协作" 按钮
   - 等待1-2分钟
   - 查看结果

### 方式2：直接输入文本

1. 跳过上传按钮
2. 直接在文本框输入简历
3. 或点击 "使用示例简历"
4. 点击 "🚀 开始AI协作"

## 📋 支持的文件格式

- ✅ PDF文件 (`.pdf`)
- ✅ Word文档 (`.docx`, `.doc`)
- ✅ 文本文件 (`.txt`)

## 🎨 按钮样式

上传按钮长这样：

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   📎 点击上传简历文件（支持PDF、Word、TXT）         │
│                                                     │
└─────────────────────────────────────────────────────┘
```

- 灰色背景
- 占满整行
- 有📎图标
- 鼠标悬停会变深色

## 🔍 如果找不到按钮

### 可能的原因：

1. **浏览器缓存问题**
   - 按 `Ctrl + F5` 强制刷新页面
   - 或清除浏览器缓存

2. **服务没有重启**
   - 关闭之前的服务
   - 重新双击 `测试上传功能.bat`

3. **代码没有更新**
   - 确认 `web_app.py` 文件已更新
   - 文件大小应该在 20KB 左右

## 🐛 调试方法

### 检查代码是否有上传功能：

打开 `web_app.py`，搜索这段代码：

```python
<input type="file" id="fileInput" accept=".pdf,.docx,.doc,.txt"
```

如果找到了，说明代码是对的。

### 检查浏览器：

1. 打开浏览器开发者工具（按 F12）
2. 查看 Console 是否有错误
3. 查看 Network 标签，看上传请求是否发送
