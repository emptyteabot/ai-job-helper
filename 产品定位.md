## 🛠️ 技术栈（2026最佳实践）

```
前端：Next.js 14 (App Router) + TypeScript
UI：Tailwind CSS + shadcn/ui + Framer Motion
后端：Vercel Serverless Functions
数据库：Supabase (Postgres + Auth + Storage)
AI：DeepSeek API (主力) + OpenAI (备用)
支付：Lemon Squeezy (自动处理税务)
分析：Plausible (隐私友好)
通知：Telegram Bot API
部署：Vercel (全球CDN)
```
