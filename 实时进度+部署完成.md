
### 手动部署
```bash
# 1. 安装Railway CLI
npm i -g @railway/cli

# 2. 登录
railway login

# 3. 初始化
railway init

# 4. 配置环境变量
railway variables set DEEPSEEK_API_KEY=your_api_key

# 5. 部署
railway up

# 6. 获取域名
railway domain
```
