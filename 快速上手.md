```

✅ AI-4 质量检查官：审核简历，指出问题

🎓 AI-5 面试教练：提供面试辅导
    ↓
```

## 🚀 立即开始测试

### 方式1：双击运行（最简单）

1. **设置API密钥**
   ```bash
   set DEEPSEEK_API_KEY=你的API密钥
   ```

2. **双击运行**
   ```

   ```


## 📋 测试流程说明

运行后，系统会：

1. **让您选择简历**
   - 可以输入自己的简历
   - 或使用示例简历（推荐首次测试）

2. **AI分析简历**
   - 提取技能、经验、项目
   - 生成简历摘要

3. **搜索匹配岗位**
   - 从模拟数据库搜索岗位
   - 按匹配度排序

4. **多AI协作优化**（核心功能）
   - 6个AI依次工作
   - 互相改进、审核、优化
   - 需要1-2分钟

5. **查看最终结果**
   - 职业分析报告
   - 推荐岗位列表
   - 优化后的简历
   - 面试准备指南
   - 模拟面试问答

## 📁 输出结果

所有结果保存在 `output/` 目录：

```
output/

├── 推荐岗位.txt           # 匹配的岗位列表
├── 优化后简历.txt         # 针对岗位优化的简历
├── 面试准备.txt           # 面试辅导建议
```

## 🤖 多AI协作机制

### 核心思想
**让AI像团队一样协作，互相辩论、改进、审核**

### 6个AI角色

| AI角色 | 任务 | 输入 | 输出 |
|--------|------|------|------|
| 职业规划师 | 分析优势和定位 | 简历 | 职业分析报告 |
| 招聘专家 | 推荐匹配岗位 | 简历+职业分析 | 推荐岗位列表 |
| 简历优化师 | 改写简历 | 简历+目标岗位 | 优化后简历v1 |
| 质量检查官 | 审核质量 | 优化后简历v1 | 问题和改进建议 |
| 简历优化师 | 再次优化 | 简历+反馈 | 优化后简历v2 |
| 面试教练 | 面试辅导 | 简历+岗位 | 面试准备指南 |
| 模拟面试官 | 模拟面试 | 简历+岗位+辅导 | 模拟面试问答 |

### 辩论流程示例

```
简历优化师: "我把项目经验改成了这样..."
    ↓
质量检查官: "有3个问题：1. 缺少量化数据 2. 技术栈不够突出 3. ..."
    ↓
简历优化师: "好的，我重新优化：1. 添加了日均5000+订单 2. 突出了Django+MySQL..."
    ↓
✅ 最终输出优化后的简历
```

## 🎨 特色功能

### 1. Markdown格式清理
自动清理DeepSeek输出中的 `*`、`**`、`###` 等符号，让结果更干净易读。

**优化前：**
```
### 职业分析
**您的核心优势**是***Python开发***...
```

**优化后：**
```
职业分析
您的核心优势是Python开发...
```

### 2. 智能匹配算法
根据技能、经验、意向计算岗位匹配度：
- 技能匹配度：70%权重
- 职位意向匹配：30%权重
- 自动排序，优先推荐最合适的

### 3. 完整辩论记录
保存所有AI的：
- 输出内容
- 推理过程
- 思考链条

方便回溯和优化。

## 🔧 常见问题

### Q1: 提示"未设置API密钥"
**A:** 运行以下命令设置：
```bash

```

### Q2: 运行很慢
**A:** 正常现象，6个AI依次工作需要1-2分钟。可以看到实时进度。
