# 🚀 AI求职助手 - 快速开始

> 用多AI协作+真实岗位抓取，帮您精准匹配工作

## ⚡ 3步开始使用

### 1️⃣ 安装（首次使用）
```bash
双击运行：一键安装.bat
```
会自动：
- ✅ 安装所有Python依赖
- ✅ 配置环境变量
- ✅ 安装OpenClaw（可选）

### 2️⃣ 配置API密钥
编辑 `.env` 文件，填入您的DeepSeek API Key：
```env
DEEPSEEK_API_KEY=sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```
[获取API Key](https://platform.deepseek.com/api_keys)

### 3️⃣ 启动服务
```bash
双击运行：启动网站.bat
```
浏览器会自动打开 `http://localhost:8000/app`

---

## 🌟 核心功能

### 📊 AI简历分析
- 6个AI角色协作分析
- 职业定位 + 岗位推荐
- 简历优化 + 面试辅导

### 🔍 真实岗位抓取（OpenClaw）
- 抓取Boss直聘真实岗位
- 一键跳转投递
- 自动记录投递历史

### 📄 多格式支持
- PDF、Word、TXT
- 图片OCR识别

---

## 🔧 OpenClaw配置（可选）

如果需要抓取真实岗位：

```bash
# 1. 测试OpenClaw
双击运行：测试OpenClaw.bat

# 2. 按提示操作
- 在Chrome中打开Boss直聘
- 点击OpenClaw扩展图标
- 点击"Attach"按钮
```

**不使用OpenClaw？** 设置 `JOB_DATA_PROVIDER=local` 使用模拟数据

## ❓ 常见问题

### Q: Pillow安装失败？
**A:** 已更新到兼容Python 3.13的版本，重新运行 `一键安装.bat`

### Q: OpenClaw是什么？
**A:** 浏览器自动化工具，用于抓取Boss直聘真实岗位。完全安全，运行在您的本机。

### Q: 不想用OpenClaw可以吗？
**A:** 可以！编辑 `.env` 设置 `JOB_DATA_PROVIDER=local` 使用本地模拟数据。

### Q: 文件夹太乱了？
**A:** 运行 `整理文件夹.bat` 自动清理重复文件。

---

## 🛠️ 故障排查

### 依赖安装失败
```bash
# 更新pip
python -m pip install --upgrade pip

# 重新安装
pip install -r requirements.txt
```

### OpenClaw未连接
```bash
# 1. 启动浏览器
openclaw browser start

# 2. 在Chrome中Attach
# 点击OpenClaw扩展图标 → Attach
```

### API调用失败
检查 `.env` 中的 `DEEPSEEK_API_KEY` 是否正确

