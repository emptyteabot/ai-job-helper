## 🐛 常见问题

### Q1: 网站还是卡顿？

**解决方案：**

1. **启动保活服务**
```bash
python keep_alive.py
```
这会每5分钟ping一次，防止Railway休眠。

2. **升级Railway套餐**
- 免费版：15分钟无请求会休眠
- 付费版：永不休眠，性能更好

3. **使用CDN加速**
- Cloudflare（免费）
- 阿里云CDN

### Q2: 自动部署失败？

**检查：**
1. GitHub Actions是否启用
2. Railway Token是否配置
3. 查看Actions日志

### Q3: 移动端显示异常？

**解决：**
1. 清除浏览器缓存
2. 强制刷新（Ctrl+F5）
3. 检查viewport设置

## 🐛 常见问题

### Q1: 网站还是卡顿？

**解决方案：**

1. **启动保活服务**
```bash
python keep_alive.py
```
这会每5分钟ping一次，防止Railway休眠。

2. **升级Railway套餐**
- 免费版：15分钟无请求会休眠
- 付费版：永不休眠，性能更好

3. **使用CDN加速**
- Cloudflare（免费）
- 阿里云CDN

### Q2: 自动部署失败？

**检查：**
1. GitHub Actions是否启用
2. Railway Token是否配置
3. 查看Actions日志

### Q3: 移动端显示异常？

**解决：**
1. 清除浏览器缓存
2. 强制刷新（Ctrl+F5）
3. 检查viewport设置
