## 🎯 核心功能

### 1. 智能投递

- **平台支持：** LinkedIn（Boss直聘开发中）
- **投递方式：** Easy Apply 自动化
- **智能问答：** AI 自动回答表单问题
- **批量处理：** 支持批量投递 1-200 个职位

### 2. 安全机制

- **反检测：** 使用 undetected-chromedriver
- **随机延迟：** 模拟人类行为（2-5秒）
- **会话持久化：** Cookie 保存，无需重复登录
- **频率限制：** 默认每次最多 200 个职位

### 3. 用户体验

- **实时进度：** WebSocket 推送投递进度
- **可视化：** 进度条、统计数字、实时日志
- **可控性：** 黑名单、白名单、提交前暂停
- **历史追踪：** 完整的投递记录

## 📈 技术亮点

### 1. 模块化设计

采用面向对象设计，易于扩展：

```python
BaseApplier (抽象基类)
    ↓
LinkedInApplier (LinkedIn 实现)
BossApplier (Boss直聘，待开发)
ZhilianApplier (智联招聘，待开发)
```

### 2. 异步任务管理

使用 `asyncio.create_task()` 实现非阻塞投递：

```python
asyncio.create_task(_run_auto_apply_task(task_id, config, jobs))
```

### 3. WebSocket 实时通信

实时推送投递进度，用户体验流畅：

```javascript
ws.onmessage = (event) => {
    const data = JSON.parse(event.data);
    updateProgress(data);
};
```

### 4. 智能问题处理

三层回答策略：

1. 预设答案匹配（快速）
2. 选项智能选择（中等）
3. AI 生成答案（高质量）
