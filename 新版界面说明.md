## 🎨 设计细节

### 颜色方案
```css
--bg-primary: #0a0a0a      /* 主背景 */
--bg-secondary: #111111    /* 卡片背景 */
--bg-tertiary: #1a1a1a     /* 次级背景 */
--text-primary: #ffffff    /* 主文字 */
--text-secondary: #a0a0a0  /* 次级文字 */
--accent: #10a37f          /* 强调色（绿色）*/
--border: #2a2a2a          /* 边框 */
```

### 字体
- **主字体**: Inter（Google Fonts）
- **备用字体**: -apple-system, BlinkMacSystemFont
- **字重**: 300-700

### 动画
- **fadeInUp**: 淡入上移（0.6s）
- **slideIn**: 滑入（0.3s）
- **pulse**: 脉冲（1.5s循环）
