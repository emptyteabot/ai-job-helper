## 🚀 API接口

### 1. 搜索岗位
```
GET /api/jobs/search?keywords=Python,Django&location=北京&limit=50
```

**参数**:
- `keywords`: 关键词（逗号分隔）
- `location`: 工作地点
- `salary_min`: 最低薪资
- `experience`: 工作经验
- `limit`: 返回数量

**返回**:
```json
{
  "success": true,
  "total": 50,
  "jobs": [
    {
      "id": "JOB000001",
      "title": "Python后端开发工程师",
      "company": "字节跳动",
      "salary": "25-50K",
      "location": "北京",
      "platform": "Boss直聘",
      "match_percentage": 95,
      ...
    }
  ]
}
```

### 2. 获取岗位详情
```
GET /api/jobs/JOB000001
```

### 3. 投递简历
```
POST /api/jobs/apply
{
  "job_id": "JOB000001",
  "resume_text": "简历内容...",
  "user_info": {}
}
```

### 4. 批量投递
```
POST /api/jobs/batch_apply
{
  "job_ids": ["JOB000001", "JOB000002"],
  "resume_text": "简历内容...",
  "user_info": {}
}
```

### 5. 数据统计
```
GET /api/statistics
```
